# Game Search - Projeto TCC

Este é o projeto web **Game Search**, desenvolvido como parte do Trabalho de Conclusão de Curso (TCC). A aplicação é uma plataforma de busca e filtragem de jogos, com foco em uma interface intuitiva e responsiva.

## 🚀 Funcionalidades

- **Busca em Tempo Real**: Encontre jogos pelo nome instantaneamente.
- **Filtragem por Categorias**: Explore jogos divididos por gêneros (Ação, Indie, RPG, etc.).
- **Carrossel de Destaques**: Visualização dinâmica dos principais jogos do momento.
- **Design Responsivo**: Interface adaptada para dispositivos móveis e desktop (Mobile-First).

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica em português brasileiro.
- **CSS3**: Estilização seguindo a metodologia BEM (Block Element Modifier) adaptada para o português.
- **JavaScript**: Lógica de manipulação do DOM, filtros e componentes dinâmicos.
- **Font Awesome**: Biblioteca de ícones.

## 📁 Estrutura do Projeto

- `index.html`: Página principal da aplicação.
- `css/estilo.css`: Arquivo de estilos (responsividade e design).
- `js/principal.js`: Lógica da aplicação e interação com o usuário.
- `img/`: Diretório destinado às imagens e ativos visuais.
